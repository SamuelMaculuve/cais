# CAIS Comunicacao & Consultoria Website

Static multi-page corporate website for CAIS, built with HTML5, Tailwind CDN utilities, custom CSS, and vanilla JavaScript.

## Creative Analysis

The inspiration references point toward premium consulting sites with editorial typography, large image-led heroes, compact navigation, calm card systems, and restrained motion. CAIS needs that polish, but with its own identity: the supplied logo uses gold as a prestige signal, while the proposal positions the company as a Mozambican point of orientation and transformation across communication, career development, and tailored consulting.

The site architecture follows the user journey: discover the promise on Home, understand the company on About, explore offers on Services, build trust through Portfolio and Process, qualify fit through Industries and FAQ, then convert on Contact.

## Design System

- Palette: charcoal, warm ivory, metallic gold, deep teal, and sage.
- Typography: Playfair Display for editorial impact, Inter for clear interface copy.
- Components: sticky navigation, hero, service cards, metric counters, timeline, FAQ accordion, case cards, contact form, footer.
- Motion: scroll reveal, counters, hover micro-interactions, mobile menu transition, and reduced-motion support.

Open `index.html` directly in a browser. Tailwind and fonts are loaded from CDNs, while project CSS/JS and images are local.
